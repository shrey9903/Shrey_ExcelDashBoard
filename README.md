# 📊 Sales Dashboard (Excel Project)

## 📌 Overview

This project is an interactive **Sales Dashboard** built using Microsoft Excel.
It provides insights into sales performance, customer behavior, and regional trends using pivot tables, advanced formulas, and dynamic visualizations.

---

## 📂 Dataset

* **Source:** Superstore Dataset (Kaggle)
* **Description:**
  The dataset contains transactional sales data including:

  * Order ID
  * Customer Name
  * Region
  * Category
  * Sales
  * Profit
  * Quantity
  * Order Date

---

## 🧹 Data Cleaning

The following preprocessing steps were performed:

* Removed duplicate records
* Handled missing values (replaced with 0 / "Unknown")
* Formatted columns (Date, Currency, Numeric)
* Ensured consistency in data types

---

## 📊 Pivot Tables

The following pivot tables were created:

### 🔹 Medium Level

* Sales by Region and Category
* Profit by Category (Sum & Average)

### 🔸 Hard Level

* Top Customers by Region (Sales-based)
* Region-wise Sales, Quantity & Average Order Value

---

## 🧮 Advanced Formulas Used

* `VLOOKUP` – Lookup values based on Order ID
* `INDEX + MATCH` – Flexible lookup method
* `IF / Nested IF` – Data classification
* `SUMIFS` – Conditional summation
* `COUNTIFS` – Conditional counting
* `AVERAGEIFS` – Conditional averaging
* `IFERROR` – Error handling

---

## 📈 Dashboard Features

* 📌 KPI Cards (Total Sales, Profit, Orders)
* 📊 Interactive Charts (Bar, Column, Pie)
* 🎛️ Slicers (Region, Category, Date)
* 📉 Customer & Regional performance insights
* 🎨 Clean and user-friendly layout

---

## 🧭 How to Use

1. Open the Excel file
2. Navigate to the **Dashboard** sheet
3. Use slicers to filter data:

   * Region
   * Category
   * Order Date
4. Charts will update dynamically
5. KPI cards reflect overall performance
